Help Manual: How to Set Up Your Search Box

We provide a search box that searches across all of your public-facing digital content. We support an unlimited number of web pages across multiple domains and we integrate your RSS feeds, tweets, YouTube videos, and Flickr and Instagram photos.

Browse through our Go-Live Punch List for some quick tips on how to configure the search experience for your website, or watch our training videos.

Activate Search

  • Code snippets
  • Search API access key*
  • Search API instructions*
  • Type-ahead API instructions*
  • i14y (Beta) API instructions*

* These APIs are available for use on official government websites only. You must be a DigitalGov Search customer. Sign in is required. Keys and instructions for each can be found under Admin Center > YourSite > Activate Search.

You can also set up a CNAME to mask your domain, which allows you to show searchers your search.YOURSITE.gov (instead of search.USA.gov).
